Experiment 05 — Cryptographic Hash
SHA-1 Hash Function
SHA-1 (Secure Hash Algorithm 1) is a cryptographic hash function that produces a 160-bit (40 hex character) digest. Designed by the NSA and published by NIST in 1995.
One-Way160-bit OutputMerkle–DamgårdDeprecated
Aim
To implement SHA-1 from scratch and understand message padding, compression rounds, and the properties of cryptographic hash functions including avalanche effect and collision resistance.
Hash Function Properties
Deterministic
Same input always produces the same hash output
Avalanche Effect
Single bit change alters approximately 50% of output bits
Pre-image Resistance
Computationally infeasible to find input given only the hash
Output Size
Always 160 bits = 20 bytes = 40 hex characters
Algorithm Overview
  1. Pad message: append 0x80, zero-pad until length ≡ 448 (mod 512), append 64-bit message length
  2. Initialize five 32-bit hash values: H0–H4 with fixed constants
  3. Process each 512-bit block: expand 16 words to 80 using XOR + left-rotate-1
  4. Run 80 rounds using three functions (Ch, Parity, Maj) with four round constants
  5. Update H0–H4 by adding the round outputs, producing the final 160-bit digest

Python — SHA-1 Implementation 
def r(n, b): return ((n << b) | (n >> (32 - b))) & 0xffffffff

def sha1(m):
    m = bytearray(m, 'utf-8'); l = len(m) * 8
    m += b'\x80'
    while (len(m) * 8) % 512 != 448: m += b'\x00'
    m += l.to_bytes(8, 'big')

    h0,h1,h2,h3,h4 = 0x67452301,0xEFCDAB89,0x98BADCFE,0x10325476,0xC3D2E1F0

    for i in range(0, len(m), 64):
        w = [int.from_bytes(m[i+j:i+j+4], 'big') for j in range(0, 64, 4)]
        for j in range(16, 80): w.append(r(w[j-3]^w[j-8]^w[j-14]^w[j-16], 1))
        a,b,c,d,e = h0,h1,h2,h3,h4

        for j in range(80):
            if j<20:   f=(b&c)|(~b&d); k=0x5A827999
            elif j<40: f=b^c^d;       k=0x6ED9EBA1
            elif j<60: f=(b&c)|(b&d)|(c&d); k=0x8F1BBCDC
            else:       f=b^c^d;       k=0xCA62C1D6
            a,b,c,d,e = (r(a,5)+f+e+k+w[j])&0xffffffff,a,r(b,30),c,d

        h0=(h0+a)&0xffffffff; h1=(h1+b)&0xffffffff
        h2=(h2+c)&0xffffffff; h3=(h3+d)&0xffffffff
        h4=(h4+e)&0xffffffff

    return '%08x%08x%08x%08x%08x' % (h0, h1, h2, h3, h4)

t = input("Text: ")
print("SHA1:", sha1(t))

Conclusion
SHA-1 demonstrates core hash function properties: determinism, avalanche effect (single bit change alters ~50% of output bits), and pre-image resistance. However, SHA-1 is now cryptographically broken — Google demonstrated a practical collision attack (SHAttered, 2017). It has been deprecated for security use; SHA-256 or SHA-3 are recommended for new systems.
