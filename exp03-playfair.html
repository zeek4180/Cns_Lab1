Playfair Cipher
A digraph substitution cipher that encrypts pairs of letters using a 5×5 key square. It was the first practical digraph substitution cipher, used by the British military in both World Wars.
SymmetricDigraph5×5 Key SquareI/J Combined
Aim
To implement the Playfair Cipher and understand digraph substitution, key square construction, and the three rules governing encryption of letter pairs.
Theory — 3 Encryption Rules
  1. Same Row: Each letter replaced by letter immediately to its right (wrapping)
  2. Same Column: Each letter replaced by letter immediately below (wrapping)
  3. Rectangle: Each letter replaced by letter on same row but in the column of the other
Key Square Construction
Size
5×5 grid using 25 letters (I and J are combined into one cell)
How to Build
Write the keyword first (no repeats), then fill remaining alphabet letters in order
Digraph Prep
Split plaintext into pairs. Insert 'X' between double letters. Pad odd-length text with 'X'
J/I Rule
All J's in plaintext are treated as I's to fit 25 letters in the 5×5 grid

def sq(k):
    k = "".join(c for c in k.upper() if c.isalpha()).replace("J", "I")
    s = []
    for c in k + "ABCDEFGHIKLMNOPQRSTUVWXYZ":
        if c not in s: s.append(c)
    return [s[i:i+5] for i in range(0, 25, 5)]

def pos(s, c):
    for r, row in enumerate(s):
        if c in row: return r, row.index(c)

def prep(t):
    t = t.upper().replace("J", "I").replace(" ", "")
    i = 0; pairs = []
    while i < len(t):
        a = t[i]
        b = t[i+1] if i+1 < len(t) else 'X'
        if a == b: b = 'X'; i += 1
        else: i += 2
        pairs.append((a, b))
    return pairs

def play(t, k, d=0):
    s = sq(k)
    p = prep(t) if not d else [t[i:i+2] for i in range(0, len(t), 2)]
    r = ""
    for a, b in p:
        ra, ca = pos(s, a); rb, cb = pos(s, b)
        if ra == rb:
            r += s[ra][(ca + (-1 if d else 1)) % 5] + s[rb][(cb + (-1 if d else 1)) % 5]
        elif ca == cb:
            r += s[(ra + (-1 if d else 1)) % 5][ca] + s[(rb + (-1 if d else 1)) % 5][cb]
        else:
            r += s[ra][cb] + s[rb][ca]
    return r

k = "PLAYFAIR EXAMPLE"
t = input("Text: ")
e = play(t, k)
print("Encoded:", e)
print("Decoded:", play(e, k, 1))

Conclusion
Playfair is significantly stronger than monoalphabetic ciphers as it encrypts digraphs (letter pairs), making simple frequency analysis ineffective. There are 676 possible digraphs vs 26 letters. However, the 25-letter key square is reconstructible from known plaintext, and modern computers can break it through hill-climbing optimization.
